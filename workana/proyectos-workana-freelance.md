# Proyectos recomendados para portfolio – Perfil Full Stack / Backend Senior (Laravel + GeoTech)

> Perfil objetivo: **Workana / Freelance internacional**  
> Enfoque: **proyectos demostrables, técnicos y vendibles**, alineados a tu CV real

## 1. Sistema Administrativo (MVP) con Laravel + Filament

**Tipo:** Proyecto estrella (core del portfolio)  
**Rol simulado:** Líder Técnico / Full Stack
**Duración estimada:** 4–6 semanas

### Descripción
Sistema administrativo completo para una empresa ficticia (inmobiliaria, logística o SaaS B2B).

### Funcionalidades clave
- Autenticación + roles y permisos (Policies / Gates)
- CRUDs complejos con Filament
- Dashboard con métricas en tiempo real
- Notificaciones con colas (Redis)
- Auditoría de acciones
- Exportaciones (CSV / Excel)

### Stack
- Laravel
- FilamentPHP
- Livewire
- PostgreSQL
- Redis
- Docker

### Qué demuestra
- Capacidad para **lanzar MVPs productivos**
- Diseño de modelo de datos
- Experiencia real en backoffice (muy demandado en Workana)

## 2. API REST de Alto Rendimiento con Laravel

**Tipo:** Proyecto técnico avanzado  
**Rol simulado:** Backend Senior

### Descripción
API para un sistema financiero / administrativo con foco en performance.

### Funcionalidades clave
- Endpoints REST versionados
- Autenticación JWT
- Caching estratégico con Redis
- Jobs y colas para procesos masivos
- Rate limiting
- Tests básicos (Feature)

### Métricas simuladas
- Reducción de tiempo de respuesta ≥ 40%
- Soporte para importaciones masivas

### Stack
- Laravel
- MySQL / PostgreSQL
- Redis
- PHPUnit
- Docker

### Qué demuestra
- Optimización real de APIs
- Escalabilidad
- Experiencia enterprise (alineado a Colegium / INDEC)

## 3. Plataforma Geoespacial Web (Geo + Backend)

**Tipo:** Proyecto diferencial (muy pocos freelancers lo tienen)  
**Rol simulado:** Líder Técnico Geo

### Descripción
Visualizador web de datos geográficos (censo, catastro o logística).

### Funcionalidades clave
- API geoespacial con PostGIS
- Servicios WMS / WFS con GeoServer
- Mapa interactivo con filtros
- Capas dinámicas
- Cache de tiles

### Stack
- Laravel
- PostgreSQL + PostGIS
- GeoServer
- OpenLayers o Leaflet
- Redis

### Qué demuestra
- Especialización **GeoTech**
- Arquitectura compleja
- Perfil altamente diferenciable en Workana
